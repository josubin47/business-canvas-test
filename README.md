# How To Run

서버와 클라이언트가 나뉘어져 있습니다. 반드시 서버를 먼저 실행하고 클라이언트 앱을 실행해주세요.

1. 저장소를 복제합니다.
2. 서버에 대한 종속성을 설치합니다.

```bash
cd server
yarn install
```

3. 클라이언트에 대한 종속성을 설치합니다.

```bash
cd client
yarn install
```

4. 서버를 시작합니다.

```bash
yarn run start-server
```

5. 클라이언트를 시작합니다.

```bash
yarn run start-client.
```

6. [http://localhost:3000/](http://localhost:3000/)에 접속 하여 앱의 동작을 확인합니다.
